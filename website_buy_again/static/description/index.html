Website Buy Again

Just Click and Buy Again. The module allows a logged in customer to view all his previously purchased products on the website. The customer can instantly view and buy products from past orders from the 'Buy Again' Menu. It eliminates the need to search the previously bought products again on the website.

Features
  •  The module adds a new menu- 'Buy Again' to the Odoo website.
  •  The menu showcases their previous purchases to the logged in customers on the website.
  •  The customers can simply add the products, they wish to purchase again, to the cart and place the order.
  •  The user can set the number of the products to be displayed to the customers.
  •  The module supports multiple websites i.e. the user can configure the module separately for each website.
  •  A Buy again slider snippet is also provided to show the purchased products to the customers via a slider on the website.
'Buy Again' Menu on the Odoo Website
  • A 'Buy Again' Menu is added to the website for the customers.
  • The customer needs to log in to his/her website account to view the 'Buy Again' menu.
Ordered products list inside menu Buy Again
  • The Buy Again Menu displays the products from their previous purchases to the customers.
Products can be added into cart
New label added with last order date and link of product
Last order details on the product page
  • The product page shows the details of the last order placed by the customer for that product.
  • The customer can click on the link to view the order in his/her website account.
Configure your module
Multiple configuration settings can be created
  • Various individual configurations can be set for different websites.
  • However, only one per website can be active at a time.
Buy Again Menu Configuration Settings
  • The user can configure the Buy Again menu in the Odoo backend.
  • The 'number of products' allows you to limit the product number displayed to the customer.
  • Using 'From Date' field, the user can choose to display those products which were purchased only after the mentioned the specified date.
Error message
  • An Error message is Displayed if the user tries to active multiple settings for the same website at the same time.
Translatable menu
Buy Now Slider Snippet
  • The user can also drag and drop Buy Now Slider Snippet anywhere on the website.
  • The snippet displays the previously ordered products via slider to the customer.
