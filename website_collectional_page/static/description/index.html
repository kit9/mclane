Odoo Website Collection Page
Products Collection Page For Your ODOO E-Commerce

Click here for the video tutorial

Features:

  • Create occasional collection pages and groups for more sale
  • SEO Management for each collection page
  • Create collection pages with some products and publish on website collection page
  • User can select the display type for each collection page on website like show banner or products
  • Create collection groups by adding different collection pages and display on website
  • User can select the display type for each collection group like single collection carousel or multi collection carousel
  • Categorize products temporarily
  • Simple and effective parameters for selecting products to be displayed on collection page
  • Publish collection page and display on website as per requirement
  • Usage of collection page URL as a hyper-link on website page
  • Positive Impact on sales figures through the use of various collection pages
